# Sanctuary — A Digital Sanctuary

A peaceful web experience for mindfulness and stillness. A place to breathe, reflect, and leave anonymous whispers.

## Features

- **Breathing Ring** — Guided 4-4-4-4 breathing animation
- **Starfield** — Twinkling stars with parallax effect
- **Whisper System** — Leave anonymous thoughts that float across the screen
- **Particles** — Ambient floating particles
- **Custom Cursor** — Blend-mode cursor that responds to hover

## Usage

Open `index.html` in any browser. No server required.

- Click to interact
- Type in the whisper box and click "Release" to add your thought to the floating whispers
- Breathe with the ring

## Tech Stack

- Pure HTML/CSS/JavaScript
- No dependencies
- LocalStorage for persisting whispers
